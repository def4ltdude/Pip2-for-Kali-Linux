# Pip2-for-Linux
Works for: Kali, Ubuntu, Mint, Debian, Arch, etc.

# Installation:
``` 
 git clone https://github.com/def4ltdude/Pip2-for-Linux.git
 cd Pip2-for-Linux
 bash pip2-install.sh 
```
Now you have pip2 in Linux.
